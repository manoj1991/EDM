Logo Sections

Logo Sections

The Logo Grid is a multi-column grid layout, ideal for showing off partner or sponsor logos.
On mobile, the logo columns go full width and stack.

Example

Customisation

You can easily change the looks by changing spacing and images.

Note that logo image widths should not exceed the parent table cell width - otherwise, the entire row can expand beyond the width of the email in some clients, such as Outlook.

  • Images — simply change the src image path, adjust dimensions and alternative text
  • Top/bottom inner spacing — change the value in the height="" attribute, as well as the font-size and line-height CSS values, on the spacer <td> at the near top and near bottom of the section
If you don't want the columns to stack on mobile, you can replace mobile-12 with some other mobile width class, such as mobile-6.
