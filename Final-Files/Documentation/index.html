The ready-to-use (Plain HTML) inline version only requires a few updates: 1. images need to be hosted online
2. paths need to be updated accordingly.
Example is as below:

 <img src="http://your-server.com/EmailBuilder/images/logo.png" width="8" height="8" />

If you are not aware to mailchimp then below is the process to import email template in your mailchimp account.

  1. Login to mailchimp website using your username & password.
  2. Click Templates from their menu & then click Create Template Button. For reference see screenshot:


  3. After clicking Create Template you will see 3 options:
    a. Paste In Code: In this process you need to copy HTML code & paste in their editor.
    b. Import HTML: You need to change all path of images & then import .html file in mailchimp
    C. Import Zip: You can zip all html, images & css and import that zip here.
